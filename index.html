Selected Projects

  • Research on cross-site scripting (XSS) based on taint tracking for Chromium v8

    Our goal is to record all of such listener functions along each data flow, and then regard them as constraints to put into a constraint solver. The constraint solver is responsible for giving a solution under all the constraints, and we can use the solution to generate exploits. Then we get relevant information about the exploit by testing different websites, and then use this information to suggest possible methods to reduce web application vulnerability.

    GitHub Repository (Code implementation of taint flow processing)

    Seek for sanitizations of JS in Chromium

  • Yoda: A Video analytic pipeline Benchmark

    Our goal is to implement a benchmark able to understand what and how video content features affect video analytic pipelines performance, to understand relative strengths & weaknesses between two video analytic pipelines in a range of settings and efficiently estimate video analytic pipelines performance on new videos.
    My major work is about to implement an interface for videostorm, a pipeline proposed in NSDI and implement an interface for awstream, a pipeline proposed in sigcomm.

    GitHub Repository
